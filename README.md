
📖 Project Description

DictionaryX (Hacker Edition) is a lightweight and powerful CLI-based Python dictionary tool that uses a free online API to fetch meanings, examples, and parts of speech for English words.
This tool is specially designed for Termux and Kali Linux users, featuring a hacker-style terminal interface with colors and borders for a more professional and engaging experience.

👉 API used: Free Dictionary API


🔹 ⚙️ Dependencies
Make sure you have the following installed:
o Python 3
o requests (for API calls)
o colorama (for terminal colors)


🔹 📦 Installation Process (Step-by-Step)
🔸 Step 1: Update packages

pkg update && pkg upgrade


🔸 Step 2: Install Python (if not installed)

pkg install python


🔸 Step 3: Install Git

pkg install git


🔸 Step 4: Clone the repository

git clone https://github.com/RupeshShide/dictionaryX


🔸 Step 5: Go to project directory

cd dictionaryX


🔸 Step 6: Install dependencies

pip install requests colorama


👉 Alternative (recommended):

python -m pip install requests colorama


🔸 Step 7: Run the tool

python dictionary.py


🔹 💻 Usage
Run the tool
Select Search Word from the menu
Enter an English word
View meanings, examples, and parts of speech


🔹 🔥 Features
🎨 Hacker-style CLI interface
